Instructions:

1. Upload an Excel, CSV, or tab-delimited text file containing your order data.

2. The application will extract the following information:

  • Customer Name
  • Customer Phone Number
  • Shipping Address (Address Line 1 and 2)
  • City, State, Zip

3. A CSV file will be generated with the proper format for shipping processing.

4. All other fields will be left empty for manual entry.
